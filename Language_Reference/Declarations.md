# 声明



### 输入输出参数

输入输出参数传递方式如下：
1. 当函数被调用时，实参的值被复制
2. 在函数体中，备份被修改
3. 当函数返回时，备份的值被赋值给原始实参

这个行为被称为*复制输入复制输出*或*用值调用*。例如，如果一个计算属性或一个有观察者的属性被当作输入输出参数传递，其getter方法的调用被当作函数调用的一部分，setter方法的调用是函数返回的一部分。

作为优化，当实参是储存在物理地址中的值，在函数体内外使用相同内存地址。优化行为被称为*用引用调用*；
